# Tic-Tac-Toe-Java

Um simples Jogo da Velha desenvolvido com Java, LWJGL, OpenGL, FreeType e GLFW.

## Informações

Desenvolvido com:

* Java
* LWJGL
* OpenGL
* FreeType
* GLFW

## Compilar

Compilar no Linux:

```
./gradlew assemble
```

**Nota:** Desenvolvido e testado somente no Linux. Não testado no Windows.
